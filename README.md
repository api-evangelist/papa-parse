# Papa ParseThis is a repo for managing machine readable artifacts for the [Papa Parse APIs](http://papaparse.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Papa Parse APIs](http://papaparse.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://papaparse.com/)- [Developer](http://papaparse.com/)- [Twitter](https://twitter.com/mholt6)- [GitHub](https://github.com/mholt/PapaParse)The [APIs.json](https://github.com/api-evangelist/papa-parse/blob/master/apis.json) index provides links to all the relevant links for the Papa Parse APIs, including the OpenAPI and Postman if they are present.
